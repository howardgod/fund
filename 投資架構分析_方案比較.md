## 三、盈餘分析：二級市場投資

### 情境：思海二級市場獲利 240 萬

#### 方案二流程

```
步驟 1：思海產生獲利 240 萬
              │
              ▼
步驟 2：依 Term Sheet 二級市場條款（50/50）
        ┌─────────────────────────────┐
        │ 創辦人分 50% = 120 萬        │
        │ 你分 50% = 120 萬            │
        └─────────────────────────────┘
              │
              ▼
步驟 3：需在 B 的股東協議中特別約定
        「二級市場獲利直接分配給 B 的股東」
              │
              ▼
步驟 4：你收到 120 萬
        • 性質：股利所得
        • 稅率：28% 分離課稅
        • 實拿：120 萬 × 72% = 86.4 萬
```

**方案二注意事項：**
- 正常情況下，思海盈餘會「全額上繳 B → B 再分配」
- 50/50 的分配方式需要在 B 的股東協議中明確約定
- 建議在協議中載明具體分配機制，避免日後認知差異

## 四、盈餘分析：績效費與管理費

### 情境：思海收取管理費 100 萬 + 績效費 200 萬

              │
              ▼

        │ B 收到 250 萬               │
        └─────────────────────────────┘
              │
              ▼

        ┌─────────────────────────────┐

**方案二說明：**

### 績效費與管理費：完整年度情境

**假設：**
- 思海管理外部資金 1 億元
- 管理費 2%/年 = 200 萬
- 當年投資報酬 30% = 3,000 萬
- 績效費 20% = 3,000 萬 × 20% = 600 萬
- 年度總收入 = 200 萬 + 600 萬 = 800 萬
- 扣除營運成本 100 萬，淨利 = 700 萬

| 分配項目 | 方案二 | 方案三 |
|---------|--------|--------|
| 思海淨利 | 700 萬 | 700 萬 |
| 分配路徑 | 思海 → B → 你 | 思海 → 你 |
| 你應得（25%） | 175 萬 | 175 萬 |
| 稅後實拿 | 126 萬 | 126 萬 |
| 架構複雜度 | 兩層 | 一層 |

## 五、虧損分析

### 情境：思海二級市場虧損 200 萬

#### 方案二 & 方案三（流程相同）

```
步驟 1：思海產生虧損 200 萬
              │
              ▼
步驟 2：依 Term Sheet 虧損補償承諾（50/50）
        ┌─────────────────────────────┐
        │ 你的帳面損失 50% = 100 萬    │
        │ 創辦人應補償 50% = 100 萬    │

              │
              ▼

              │
              ▼

        • 收到補償：100 萬（創辦人的 50%）
        • 淨損失：0 元
```

## 六、第三方入股應對

### 情境：第三方 C 想投 2000 萬買思海 25%

#### 方案二：透過 B 間接行使權利

```
┌─────────────────────────────────────────────────────────┐
│                      決策流程                            │
├─────────────────────────────────────────────────────────┤
│                                                         │
│  1. 創辦人提出：C 要入股思海                             │
│                  │                                      │
│                  ▼                                      │
│  2. 觸發「重大事項否決權」                               │
│     條款：思海股權變動需經 B 全體股東同意                │
│     說明：你的權利在 B 的層級行使                        │
│                  │                                      │
│          ┌──────┴──────┐                                │
│          ▼             ▼                                │
│       你否決         你同意                              │
│          │             │                                │
│          ▼             ▼                                │
│       交易終止      進入下一步                           │
│                        │                                │
│                        ▼                                │
│  3. 觸發「優先認購權」                                   │
│     你可認購：25%（你在B的持股）× 25%（C要買的）         │
│            = 6.25% 的優先認購權                         │
│                        │                                │
│          ┌─────────────┴─────────────┐                  │
│          ▼                           ▼                  │
│       你認購                       你放棄                │
│          │                           │                  │
│          ▼                           ▼                  │
│    維持間接持股比例              被稀釋                  │
│                                      │                  │
│                                      ▼                  │
│  4. 觸發「反稀釋保護」                                   │
│     若 C 入股價 < 60萬/1%                               │
│     → 在 B 的層級調整你的持股比例                        │
│                                                         │
└─────────────────────────────────────────────────────────┘
```

**方案二說明：**
- 權利在 B 的層級行使
- 需在 B 的股東協議中明確約定對思海的控制條款

**建議在 B 的股東協議明確寫入：**
> 「B 公司對思海行使股東權利時（包括但不限於同意思海增資、減資、股權轉讓、合併、重大資產處分），需經 B 公司全體股東書面同意。」

│          ┌─────────────┴─────────────┐                  │

### 第三方入股：稀釋比較

**情境：C 投 2000 萬買思海 25%，你選擇不認購**

| 項目 | 方案二 | 方案三 |
|------|--------|--------|

## 七、綜合比較

| 項目 | 方案二（透過 B） | 方案三（直接持有） |
|------|-----------------|-------------------|
| **持股關係** | B 的股東（間接持有思海） | 思海的直接股東 |
| **二級市場盈餘** | 需在協議中約定分配方式 | 直接分配 |
| **績效費管理費** | 思海 → B → 你（兩層） | 思海 → 你（一層） |
| **虧損補償** | 相同機制 | 相同機制 |
| **稅負** | 28% 股利稅 | 28% 股利稅 |

| **優先認購權** | 透過 B 間接行使 | 直接行使 |
| **反稀釋保護** | 在 B 層級調整 | 在思海層級調整 |
| **法律適用** | 需依賴協議條款完整性 | 公司法直接適用 |
| **架構複雜度** | 較高（兩間公司、兩層協議） | 較低（一間公司） |

## 八、結論與建議

### 兩方案特點比較

| 項目 | 方案二 | 方案三 |
|---------|--------|--------|
| 盈餘分配路徑 | 兩層 | 一層 |
| 權利行使方式 | 間接 | 直接 |
| 協議條款依賴度 | 較高 | 較低 |
| 架構複雜度 | 較高 | 較低 |

### 最終建議

**方案三（直接持有思海）架構較為單純，權利義務關係明確。**

若有其他商業考量需採用方案二，建議確保以下條款納入 B 的股東協議：

---

### 若採用方案二，建議納入的條款

1. **二級市場盈餘分配機制**
   > 「思海二級市場投資之損益，應依 Term Sheet 約定之比例（50/50），直接分配予 B 公司之股東，不適用一般盈餘上繳及分配程序。」

2. **思海重大事項決策機制**
   > 「B 公司對思海行使股東權利時（包括但不限於同意思海增資、減資、股權轉讓、合併、分割、重大資產處分、修改章程），需經 B 公司全體股東書面同意。」

3. **盈餘分配時程**
   > 「思海每年度之可分配盈餘，應於年度結束後 90 日內，上繳 B 公司並分配予股東。」

4. **資訊揭露**
   > 「B 公司應確保投資人得查閱思海之所有財務紀錄、帳簿及相關文件。」
