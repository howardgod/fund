## 一、架構概述

### 方案一：直接持有思海 25%，但透過勞務合約取得收益

```
┌─────────────────────────────────────┐
│         控股公司 B                   │
│      （創辦人 100% 持有）            │
│              │                      │
│              ▼ 75% 持股             │
│  ┌─────────────────────┐            │
│  │       思海           │            │
│  │  ┌───────┬───────┐  │            │
│  │  │ B 75% │ 你 25% │  │            │
│  │  └───────┴───────┘  │            │
│  └─────────────────────┘            │
│              │                      │
│              ▼ 盈餘上繳 B            │
│         控股公司 B                   │
│              │                      │
│              ▼ 勞務合約              │
│  ┌─────────────────────┐            │
│  │        你            │            │
│  │  （收益透過勞務合約） │            │
│  └─────────────────────┘            │
└─────────────────────────────────────┘
```

**你的身份**：思海 25% 股東，但收益透過 B 的勞務合約取得（非以股東身份分配）

**核心問題**：你是股東，但收益管道是勞務合約，兩者分離。

---

### 方案二：透過控股公司 B 持有股權

```
┌─────────────────────────────────────┐
│           控股公司 B                 │
│  ┌───────────┬───────────┐          │
│  │ 創辦人 75% │   你 25%   │          │
│  └───────────┴───────────┘          │
│              │                      │
│              ▼ 100% 持股            │
│  ┌─────────────────────┐            │
│  │       思海           │            │
│  │   （操盤公司）        │            │
│  └─────────────────────┘            │
└─────────────────────────────────────┘
```

**你的身份**：B 的股東（間接持有思海 25%），以股東身份分配收益

---

### 方案三：直接持有思海股權

```
┌─────────────────────────────────────┐
│             思海                     │
│  ┌───────────┬───────────┐          │
│  │ 創辦人 75% │   你 25%   │          │
│  └───────────┴───────────┘          │
└─────────────────────────────────────┘
```

**你的身份**：思海的直接股東，以股東身份分配收益

## 二、持股與法律關係

| 項目 | 方案一（勞務合約） | 方案二（透過 B） | 方案三（直接持有） |
|------|-------------------|-----------------|-------------------|
| 你持有思海的股份 | 25%（直接） | 0%（間接 25%） | 25%（直接） |
| 你持有 B 的股份 | 0% | 25% | N/A |
| B 持有思海的股份 | 75% | 100% | N/A |
| 創辦人持有 B | 100% | 75% | N/A |
| **收益取得方式** | 勞務合約（從 B） | 股利分配（從 B） | 股利分配（從思海） |
| **收益所得性質** | 勞務所得 | 股利所得 | 股利所得 |
| **適用法律** | 公司法 + 民法契約 | 公司法 | 公司法 |

---

### 方案一的特殊問題

```
┌─────────────────────────────────────────────────────────┐
│                  方案一的架構矛盾                        │
├─────────────────────────────────────────────────────────┤
│                                                         │
│  你的股東身份            你的收益來源                    │
│  ┌─────────────┐         ┌─────────────┐               │
│  │ 思海 25%    │         │ 控股公司 B   │               │
│  │ 直接股東    │         │ 勞務合約    │               │
│  └─────────────┘         └─────────────┘               │
│         │                       │                      │
│         ▼                       ▼                      │
│  股東權益歸你            收益從 B 來                    │
│  （價值上升你享有）      （性質為勞務所得）              │
│                                                         │
│  問題：股東身份與收益管道分離，產生以下風險：            │
│  1. 勞務所得稅率 40%，高於股利所得 28%                  │
│  2. 勞務合約可被終止，但股東權益不能                    │
│  3. 第三方入股時，股權價值上升，但勞務合約收益不一定調整 │
│                                                         │
└─────────────────────────────────────────────────────────┘
```

## 三、盈餘分析：二級市場投資

### 情境：思海二級市場獲利 240 萬

#### 方案一流程

```
步驟 1：思海產生獲利 240 萬
              │
              ▼
步驟 2：正常情況下應按股權分配
        • B（75%）應得 180 萬
        • 你（25%）應得 60 萬（股利）
              │
              ▼
步驟 3：但實際執行是透過勞務合約
        • 思海盈餘上繳 B
        • B 依勞務合約支付給你
              │
              ▼
步驟 4：依 Term Sheet 二級市場條款（50/50）
        B 透過勞務合約支付 50% 給你
        ┌─────────────────────────────┐
        │ 你收到 120 萬（勞務報酬）    │
        │ 創辦人保留 120 萬           │
        └─────────────────────────────┘
              │
              ▼
步驟 5：你收到 120 萬
        • 性質：勞務所得（執行業務所得）
        • 稅率：併入綜所稅，最高 40%
        • 若適用 40% 稅率，實拿：120 萬 × 60% = 72 萬
```

---

```
步驟 1：思海產生獲利 240 萬
              │
              ▼
步驟 2：依 Term Sheet 二級市場條款（50/50）
        ┌─────────────────────────────┐
        │ 創辦人分 50% = 120 萬        │
        │ 你分 50% = 120 萬            │
        └─────────────────────────────┘
              │
              ▼
步驟 3：透過 B 分配股利給你
              │
              ▼
步驟 4：你收到 120 萬
        • 性質：股利所得
        • 稅率：28% 分離課稅
        • 實拿：120 萬 × 72% = 86.4 萬
```

---

#### 方案三流程

```

              │
              ▼

        ┌─────────────────────────────┐
        │ 創辦人分 50% = 120 萬        │
        │ 你分 50% = 120 萬            │
        └─────────────────────────────┘

步驟 3：思海直接分配股利給你

## 四、盈餘分析：績效費與管理費

**假設：**
- 思海管理外部資金 1 億元
- 管理費 2%/年 = 200 萬

- 績效費 20% = 3,000 萬 × 20% = 600 萬
- 年度總收入 = 200 萬 + 600 萬 = 800 萬

## 七、第三方入股應對

### 情境：第三方 C 想投 2000 萬買思海 25%

#### 方案一：股權與收益分離的問題

```
┌─────────────────────────────────────────────────────────┐
│                   C 入股前                               │
├─────────────────────────────────────────────────────────┤
│                                                         │
│  思海股權結構              你的收益來源                  │
│  ┌─────────────┐          ┌─────────────┐              │
│  │ B：75%      │          │ 控股公司 B   │              │
│  │ 你：25%     │          │ 勞務合約    │              │
│  └─────────────┘          │ 給你 25%    │              │
│                           └─────────────┘              │
│                                                         │
└─────────────────────────────────────────────────────────┘

                        ▼ C 入股 25%

┌─────────────────────────────────────────────────────────┐
│                   C 入股後                               │
├─────────────────────────────────────────────────────────┤
│                                                         │
│  思海股權結構              你的收益來源                  │
│  ┌─────────────┐          ┌─────────────┐              │
│  │ B：56.25%   │          │ 控股公司 B   │              │
│  │ 你：18.75%  │          │ 勞務合約    │              │
│  │ C：25%      │          │ ???         │              │
│  └─────────────┘          └─────────────┘              │
│                                                         │
│  你的股權變化：                                          │
│  • 股權從 25% 稀釋到 18.75%                             │
│  • 但股權價值可能上升（C 帶來資金）                      │
│                                                         │
│  你的收益變化：                                          │
│  • 收益來自 B 的勞務合約                                │
│  • B 持有思海從 75% 降到 56.25%                         │
│  • B 收到的盈餘減少                                     │
│  • 若勞務合約沒有調整，你的收益可能不變或減少            │
│                                                         │
│  問題：                                                  │
│  • 股權價值上升，但收益不一定跟著上升                    │
│  • 需確認勞務合約的計算基礎                              │
│                                                         │
└─────────────────────────────────────────────────────────┘
```

**方案一的特殊風險：**

| 項目 | 說明 |
|------|------|
| 股權價值 | 你是思海股東，C 入股後公司價值上升，你的股權價值也上升 ✓ |
| 收益來源 | 你的收益來自 B 的勞務合約，不是股東分配 |
| 收益計算 | 若勞務合約約定「思海盈餘的 25%」→ 收益不變 |
| | 若勞務合約約定「B 收到盈餘的某比例」→ B 收到的減少，你的收益也減少 |
| 調整需求 | C 入股後，需確認 B 是否調整勞務合約 |

│                  │                                      │
│          ┌──────┴──────┐                                │
│          ▼             ▼                                │
│       你否決         你同意                              │
│          │             │                                │
│          ▼             ▼                                │

│                        │                                │
│                        ▼                                │

│          ┌─────────────┴─────────────┐                  │

│       你認購                       你放棄                │

│          ▼                           ▼                  │
│    維持間接持股比例              被稀釋                  │

│                                      ▼                  │
│  4. 觸發「反稀釋保護」                                   │
│     若 C 入股價 < 60萬/1%                               │

│                                                         │
└─────────────────────────────────────────────────────────┘
```

#### 方案三：直接行使權利

### 方案一的主要問題

1. **股權與收益分離**
   - 你是思海股東，但收益透過 B 的勞務合約
   - 第三方入股時，股權價值上升，但收益不一定跟著調整

2. **稅務不利**
   - 勞務所得稅率最高 40%
   - 股利所得稅率 28%
   - 每年稅後差異約 35 萬

3. **合約終止風險**
   - 勞務合約可能被終止或不續約
   - 但你的股東身份不受影響
   - 造成「有股權但沒收益」的情況

### 若採用方案二，建議納入的條款

1. **二級市場盈餘分配機制**
   > 「思海二級市場投資之損益，應依 Term Sheet 約定之比例（50/50），直接分配予 B 公司之股東，不適用一般盈餘上繳及分配程序。」

2. **思海重大事項決策機制**
   > 「B 公司對思海行使股東權利時（包括但不限於同意思海增資、減資、股權轉讓、合併、分割、重大資產處分、修改章程），需經 B 公司全體股東書面同意。」

3. **盈餘分配時程**
   > 「思海每年度之可分配盈餘，應於年度結束後 90 日內，上繳 B 公司並分配予股東。」

4. **資訊揭露**
   > 「B 公司應確保投資人得查閱思海之所有財務紀錄、帳簿及相關文件。」

---

### 若採用方案一，建議確認事項

1. **勞務合約收益計算基礎**
   - 確認是「思海盈餘的 25%」還是「B 收到盈餘的某比例」
   - 前者較有利，後者會隨 B 持股變動

2. **第三方入股時的調整機制**
   - 約定當有第三方入股時，勞務合約應如何調整
   - 確保股權價值上升時，收益也跟著上升

3. **合約期限與終止條件**
   - 約定與投資期間相同的合約期限
   - 明確終止條件及違約處理
