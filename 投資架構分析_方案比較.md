## 一、架構概述

### 方案一：持有思海 25% 股權，但透過勞務合約取得收益

```
┌─────────────────────────────────────┐
│         控股公司 B                   │
│      （創辦人 100% 持有）            │
│              │                      │
│              ▼ 100% 持股            │
│  ┌─────────────────────┐            │
│  │       思海           │            │
│  │   （操盤公司）        │            │
│  │                      │            │
│  │  你投資 1500 萬       │            │
│  │  目標：25% 股權       │            │
│  └─────────────────────┘            │
│              │                      │
│              ▼ 每月盈餘上繳          │
│         控股公司 B                   │
│              │                      │
│              ▼ 勞務合約              │
│  ┌─────────────────────┐            │
│  │        你            │            │
│  │  （拿 25% 收益）      │            │
│  │  （收益性質：勞務報酬）│            │
│  └─────────────────────┘            │
└─────────────────────────────────────┘
```

**你的身份**：投資了 1500 萬，但收益透過勞務合約取得（非以股東身份分配）

**核心問題**：Term Sheet 約定你持有 25% 股權，但實際收益是透過勞務合約，法律上這是兩件事。

---

### 方案二：透過控股公司 B 持有股權

```
┌─────────────────────────────────────┐
│           控股公司 B                 │
│  ┌───────────┬───────────┐          │
│  │ 創辦人 75% │   你 25%   │          │
│  └───────────┴───────────┘          │
│              │                      │
│              ▼ 100% 持股            │
│  ┌─────────────────────┐            │
│  │       思海           │            │
│  │   （操盤公司）        │            │
│  └─────────────────────┘            │
└─────────────────────────────────────┘
```

**你的身份**：B 的股東（間接持有思海 25%），以股東身份分配收益

---

### 方案三：直接持有思海股權

```
┌─────────────────────────────────────┐
│             思海                     │
│  ┌───────────┬───────────┐          │
│  │ 創辦人 75% │   你 25%   │          │
│  └───────────┴───────────┘          │
└─────────────────────────────────────┘
```

**你的身份**：思海的直接股東，以股東身份分配收益

## 二、持股與法律關係

| 項目 | 方案一（勞務合約） | 方案二（透過 B） | 方案三（直接持有） |
|------|-------------------|-----------------|-------------------|
| 你的投資金額 | 1,500 萬 | 1,500 萬 | 1,500 萬 |
| 你應得的股權 | 25% | 25% | 25% |
| 你實際持有思海股份 | 0%（B 持有 100%） | 0%（間接 25%） | 25% |
| 你持有 B 的股份 | 0% | 25% | N/A |
| 收益取得方式 | 勞務合約 | 股利分配 | 股利分配 |
| **你的法律身份** | 勞務提供者 | 股東 | 股東 |
| **適用法律** | 民法（契約） | 公司法 | 公司法 |
| **收益所得性質** | 勞務所得 | 股利所得 | 股利所得 |

## 三、盈餘分析：二級市場投資

### 情境：思海二級市場獲利 240 萬

#### 方案一流程

```
步驟 1：思海產生獲利 240 萬
              │
              ▼
步驟 2：思海將盈餘上繳給 B
        ┌─────────────────────────────┐
        │ B 收到 240 萬               │
        └─────────────────────────────┘
              │
              ▼
步驟 3：依 Term Sheet 二級市場條款（50/50）
        B 透過勞務合約支付 50% 給你
        ┌─────────────────────────────┐
        │ 你收到 120 萬（勞務報酬）    │
        │ 創辦人保留 120 萬           │
        └─────────────────────────────┘
              │
              ▼
步驟 4：你收到 120 萬
        • 性質：勞務所得（執行業務所得）
        • 稅率：併入綜所稅，最高 40%
        • 若適用 40% 稅率，實拿：120 萬 × 60% = 72 萬
```

---

```
步驟 1：思海產生獲利 240 萬
              │
              ▼
步驟 2：依 Term Sheet 二級市場條款（50/50）
        ┌─────────────────────────────┐
        │ 創辦人分 50% = 120 萬        │
        │ 你分 50% = 120 萬            │
        └─────────────────────────────┘
              │
              ▼
步驟 3：透過 B 分配股利給你
              │
              ▼
步驟 4：你收到 120 萬
        • 性質：股利所得
        • 稅率：28% 分離課稅
        • 實拿：120 萬 × 72% = 86.4 萬
```

---

#### 方案三流程

```
步驟 1：思海產生獲利 240 萬
              │
              ▼

        ┌─────────────────────────────┐
        │ 創辦人分 50% = 120 萬        │
        │ 你分 50% = 120 萬            │
        └─────────────────────────────┘
              │
              ▼
步驟 3：思海直接分配股利給你
              │
              ▼
步驟 4：你收到 120 萬
```

## 四、盈餘分析：績效費與管理費

### 完整年度情境

**假設：**
- 思海管理外部資金 1 億元
- 管理費 2%/年 = 200 萬
- 當年投資報酬 30% = 3,000 萬
- 績效費 20% = 3,000 萬 × 20% = 600 萬
- 年度總收入 = 200 萬 + 600 萬 = 800 萬
- 扣除營運成本 100 萬，淨利 = 700 萬

---

#### 方案一流程

```
步驟 1：思海淨利 700 萬
              │
              ▼

        │ B 收到 700 萬               │
        └─────────────────────────────┘
              │
              ▼
步驟 3：B 依勞務合約支付 25% 給你
        ┌─────────────────────────────┐

        │ 創辦人保留 525 萬           │
        └─────────────────────────────┘

              ▼
```

## 六、虧損分析

### 情境：思海二級市場虧損 200 萬

#### 方案一

```
步驟 1：思海產生虧損 200 萬
              │
              ▼
步驟 2：虧損補償機制
        • 若勞務合約有約定 50/50 虧損分擔：
          創辦人應補償你 100 萬
        • 若勞務合約未約定：
          無補償機制
              │
              ▼
步驟 3：問題
        • 勞務合約的效力不如股東協議
        • 虧損補償的執行依賴合約條款完整性
        • 你的投資（1500 萬）在思海，但你不是股東，
          難以主張股東權益
```

---

#### 方案二 & 方案三

```
步驟 1：思海產生虧損 200 萬
              │
              ▼
步驟 2：依 Term Sheet 虧損補償承諾（50/50）
        ┌─────────────────────────────┐
        │ 你的帳面損失 50% = 100 萬    │
        │ 創辦人應補償 50% = 100 萬    │
        └─────────────────────────────┘
              │
              ▼
步驟 3：創辦人 30 日內私人匯款 100 萬給你
              │
              ▼
步驟 4：你的實際損失
        • 帳面損失：100 萬（你的 50%）
        • 收到補償：100 萬（創辦人的 50%）
        • 淨損失：0 元
```

---

### 虧損處理比較

| 項目 | 方案一 | 方案二 | 方案三 |
|------|--------|--------|--------|
| 虧損補償依據 | 勞務合約（若有約定） | 股東協議 + Term Sheet | 股東協議 + Term Sheet |
| 法律保障強度 | 較弱（民法契約） | 較強（公司法） | 較強（公司法） |
| 你的身份 | 勞務提供者 | 股東 | 股東 |
| 主張權益的基礎 | 合約債權 | 股東權益 | 股東權益 |

## 七、第三方入股應對

### 情境：第三方 C 想投 2000 萬買思海 25%

#### 方案一：股東權利不明確

```
┌─────────────────────────────────────────────────────────┐
│                      決策流程                            │
├─────────────────────────────────────────────────────────┤
│                                                         │
│  1. 創辦人提出：C 要入股思海                             │
│                  │                                      │
│                  ▼                                      │
│  2. 你的權利                                            │
│     • 你投資了 1500 萬，應該有 25% 股權                  │
│     • 但實際上 B 持有思海 100%                          │
│     • 你的收益來源是勞務合約，不是股東身份               │
│                  │                                      │
│                  ▼                                      │
│  3. 問題                                                │
│     • 否決權：需看勞務合約是否有約定                     │
│     • 優先認購權：需看勞務合約是否有約定                 │
│     • 反稀釋保護：需看勞務合約是否有約定                 │
│     • 即使有約定，執行力度不如股東協議                   │
│                  │                                      │
│                  ▼                                      │
│  4. 風險                                                │
│     • 若 C 入股，思海價值提升                           │
│     • 但你不是股東，無法直接享受價值提升                 │
│     • 你的收益仍然依賴勞務合約                          │
│                                                         │
└─────────────────────────────────────────────────────────┘
```

---

#### 方案二：透過 B 間接行使權利

```
┌─────────────────────────────────────────────────────────┐
│                      決策流程                            │
├─────────────────────────────────────────────────────────┤
│                                                         │
│  1. 創辦人提出：C 要入股思海                             │
│                  │                                      │
│                  ▼                                      │
│  2. 觸發「重大事項否決權」                               │
│     條款：思海股權變動需經 B 全體股東同意                │
│     說明：你的權利在 B 的層級行使                        │
│                  │                                      │
│          ┌──────┴──────┐                                │
│          ▼             ▼                                │
│       你否決         你同意                              │
│          │             │                                │
│          ▼             ▼                                │
│       交易終止      進入下一步                           │
│                        │                                │
│                        ▼                                │
│  3. 觸發「優先認購權」                                   │
│     你可認購：25%（你在B的持股）× 25%（C要買的）         │
│            = 6.25% 的優先認購權                         │
│                        │                                │
│          ┌─────────────┴─────────────┐                  │
│          ▼                           ▼                  │
│       你認購                       你放棄                │
│          │                           │                  │
│          ▼                           ▼                  │
│    維持間接持股比例              被稀釋                  │
│                                      │                  │
│                                      ▼                  │
│  4. 觸發「反稀釋保護」                                   │
│     若 C 入股價 < 60萬/1%                               │
│     → 在 B 的層級調整你的持股比例                        │
│                                                         │
└─────────────────────────────────────────────────────────┘
```

#### 方案三：直接行使權利

```
┌─────────────────────────────────────────────────────────┐

│     條款：思海股權變動需經你書面同意                     │
│     說明：你直接行使權利                                 │

│          ┌──────┴──────┐                                │
│          ▼             ▼                                │
│       你否決         你同意                              │
│          │             │                                │
│          ▼             ▼                                │
```

### 方案一的主要問題

1. **法律身份不一致**
   - Term Sheet 約定你持有 25% 股權
   - 但實際收益透過勞務合約取得
   - 法律上你不是股東

2. **稅務不利**
   - 勞務所得稅率最高 40%
   - 股利所得稅率 28%
   - 每年稅後差異約 35 萬

3. **權利保障較弱**
   - 股東權利（否決、認購、反稀釋）需依賴合約約定
   - 合約效力不如公司法保護的股東權益

4. **合約終止風險**
   - 勞務合約可能被終止或不續約
   - 股東權益不能被單方面剝奪
